# TMolecule Seasonal Presets

Each preset folder contains CSS overrides for band colors, CTA buttons, and accents.

## How to apply a preset

1. Open Shopify Admin → Themes → Edit code
2. Open `assets/tmolecule-product.css`
3. Find the `/* SEASONAL OVERRIDE */` section at the bottom (or add it)
4. Paste the preset's `override.css` content there
5. Save → done. Page instantly reflects the season.

## How to revert

Delete everything below `/* SEASONAL OVERRIDE */` in the CSS file. Save.
