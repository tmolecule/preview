Our approach

Three pillars on every page.

1 · Structure

Make content extractable by AI engines — schema, semantic HTML, FAQ blocks, comparison tables, speakable selectors, markdown alternatives.

2 · Authority

Make every claim citable — peer-reviewed sources inline, Person author with E-E-A-T signals, freshness dates, expert framing.

3 · Presence

Be where AI crawlers look — explicit IETF Content-Signal opt-in, /llms.txt, /.md, /manifest.json, allow-lists for 11 named AI bots.

The signature pattern

Tradition + Science.

Every health claim pairs the classical source with the peer-reviewed study.

❖ Ayurveda's logic

Classical Ayurveda treats the trikaṭu stack — ginger, black pepper, long pepper — as dīpana-pācana: agni-kindling and absorption-enhancing. Charaka Samhita prescribed it 2,000+ years ago.

◦ What pharmacology measures

Shoba et al. 1998 (Planta Medica) found piperine — the alkaloid in black pepper — increased curcumin bioavailability in human volunteers by 2,000%. The same multiplier mechanism Ayurveda was using two millennia earlier.

Live on the masala-chai-recipe page. Five articles use this pattern; pattern documented in seed/_README.md.

Under the hood

The stack.

Edge

Cloudflare Worker rendering KV-backed JSON content. Sub-100ms cold-start globally. Routes: HTML article, .md alternative, /llms.txt, /manifest.json, /sitemap.xml, /robots.txt, /health.

Content

Each article = one JSON file in seed/. Single command pushes to KV. Fields: title, h1, meta_description, body_html, image_url, faqs, sources, keywords, dates.

Schema

JSON-LD generated per page from the seed JSON. Article + FAQPage + Speakable + BreadcrumbList + citation[] all live, validated by Google Rich Results Test.

What "3 mentions" actually means

The number is from an AI-citation monitoring snapshot — sampled US-English LLM queries, what was caught at that point in time. It's a floor, not a ceiling. Citations in queries outside the sample, other geographies, or post-snapshot changes don't appear here. The same snapshot re-run returned the same 3 — data is stable, sample is finite.

The strategic insight

All 3 citations are AI engines pulling tmolecule.com into nutrition / calorie database answers — co-cited with eatthismuch.com, menus.princeton.edu, workwell.usc.edu, specialolympics.ca. The brand is read as "macro source," not "tea authority." The Learn site deploy targets the gap.
